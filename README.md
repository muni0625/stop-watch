# ⏱️ Stopwatch Web App

A simple and elegant stopwatch web application built using **HTML**, **CSS**, and **JavaScript**. This app can **start**, **pause**, **reset**, and **record lap times** with a user-friendly interface.

---

## 🔧 Features

- ✅ Start and Pause Timer
- 🔁 Reset Timer
- 📍 Record multiple Lap Times
- 🎨 Modern and responsive UI
- 🕒 Precise time tracking (Hours:Minutes:Seconds:Milliseconds)
